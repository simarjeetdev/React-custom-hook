##  Project Overview

This project demonstrates how to create and use a **custom React hook (`useFetch`)** to fetch data from an external API and display it on the UI using **reusable components and props**.

The main goal of this assignment is to:
- Abstract API logic into a custom hook
- Handle **loading**, **error**, and **success** states properly
- Render fetched data in **card-based UI components**

---

## What This Project Covers

- Custom Hook creation (`useFetch`)
- API data fetching using `fetch`
- State management using `useState`
- Side effects using `useEffect`
- Component-based UI
- Props usage for data flow
- Conditional rendering (Loading / Error / Data)
